Overview

Changing Soon - Checkback for changes later

Update is a tool that runs on all supported workstations as a check in remote management. It allows for easy automatic installation and updating of packages.

Details

Update is written in Python. It is an "old" system, so the source is not available on GitLab.

Debugging

Update rarely fails, and is safe to interrupt and later resume at any point during execution. Generally the only case where you will see a failure is when the workstation's internet connection is slow or unstable.

You may also see a message referencing a lock file "C:\Cilix\Update\lock". If the check fails more than one with this message then delete the lock file.
