Overview

PCSetup is a tool that can be manually run to perform most of the work required when setting up a new workstation. The tool itself requires PCBootstrap to be run first, but a standalone binary PCSetup-AutoRun is downloadable from the Portal which runs PCBootstrap and then PCSetup automatically.

Details

PCSetup is written in Python. It is an "old" system, so the source is not available on GitLab.

Debugging

PCSetup rarely fails, and is safe to interrupt and later resume at any point during execution. Generally the only case where you will see a failure is when the workstation's internet connection is slow or unstable.
