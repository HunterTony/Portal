Core Tools - Setup

{% endblock %} {% block content %}

Overview

Setup is a tool that can be manually run to perform most of the work required when setting up a new workstation. The tool itself requires Bootstrap to be run first, but the Setup Workstation binary does this automatically.

Details

Setup is written in Python, and the source is hosted on GitLab under "Portal/root/tools/setup_workstation/binary".

You can add a new action to Setup by creating a new Python script under "Portal/root/tools/setup_workstation/actions". Use one of the existing scripts as a template.

Packages for Setup can be configured using the Package Manager tool.

Debugging

Setup rarely fails, and is safe to interrupt and later resume at any point during execution. Generally the only case where you will see a failure is when the workstation's internet connection is slow or unstable.
