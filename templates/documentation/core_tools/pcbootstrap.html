Overview

PCBootstrap is responsible for installing the prerequisites required for pretty much everything else to work. It is one of the only tools written in Powershell because it is most often run on a new workstation which wont yet have Python installed.

PCBootstrap installs the following packages:

  • Visual Studio 2010 Redistributable - Required by the Python interpreter.
  • Python 3.4 Interpreter - Required by pretty much everything else.
  • ActiveTcl - A Tcl interpreter. Required by TkInter, which is used by PCSetup and various other tools to create forms.
  • PyCurl - A Python library used for HTTP and FTP requests. Used by PCSetup and various other tools to download and upload from HTTP and FTP servers.

Details

PCBootstrap is written to be compatible with Powershell 2.0 and above, which is available on every Windows OS after XP. XP support is theoretically possible by manually installing Powershell 2.0, but this is not advised as other tools are not designed to support XP.

PCBootstrap is run automatically by PCSetup-AutoRun before PCSetup is run.

PCBootstrap is available as an automated task in Remote Management, which can be run in the background on any existing workstations that are enrolled (for example if a new client joins).

The source code for PCBootstrap is hosted on GitLab under Remote Management/Core.

Debugging

PCBootstrap rarely fails, and is safe to interrupt and later resume at any point during execution. Generally the only case where you will see a failure is when the workstation's internet connection is slow or unstable.
