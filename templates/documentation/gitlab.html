Overview

GitLab is a hosted Git platform that houses source code for all modern systems. By "modern" I mean any projects written or updated in the end of 2015 and the beginning of 2016.

Details

Each project is owned by the "cilix" group. The group is owned by a shared user "cilix_support", the password for which can be found in PasswordSafe.

Anyone who wishes to access GitLab should create an account of their own and invite it to the group using the cilix_support account.


If you need to make a change to any of the projects, you will need to:

  • Pull the project
  • Make your changes to the source files
  • Commit your changes
  • Push your commits

Detailed information on these steps can be found on the Git Website
