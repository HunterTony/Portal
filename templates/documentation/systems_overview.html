Core Tools

The trifecta of configuration and maintainence tools. Bootstrap, Setup, and Update.

Bootstrap is responsible for installing the prerequisites required for pretty much everything else to work. It is one of the only tools written in Powershell because it is most often run on a new workstation which wont yet have Python installed.

Setup is a tool that can be manually run to perform most of the work required when setting up a new workstation. The tool itself requires Bootstrap to be run first, but the Setup Workstation binary does this automatically.

Update is a tool that runs on all supported workstations as a check in Remote Management. It allows for easy automatic installation and updating of packages.

Remote Management Scripts

Checks and tasks that run on the Remote Management platform.

Checks are divided into these categories:

  • Alert - Monitor various bits of a workstation and fail should they detect an issue.
  • Information - Gather information about a workstation and display it. These do not fail during normal operation.

Tools are exactly the same as Checks, but run as Background Tasks instead. This allows them to have a maximum 24 hour runtime, rather than the 150 second maximum for checks.
