Core Tools

The trifecta of PC configuration and maintainence tools. PCBootstrap, PCSetup, and PCUpdate.

PCBootstrap is responsible for installing the prerequisites required for pretty much everything else to work. It is one of the only tools written in Powershell, because before it is run the Python interpreter is not installed.

PCSetup is a tool that can be manually run to perform most of the work required when setting up a new workstation. The tool itself requires PCBootstrap to be run first, but a standalone binary PCSetup-AutoRun is downloadable from the Portal which runs PCBootstrap and then PCSetup automatically.

PCUpdate is a tool that runs on all supported workstations as a check in Remote Management. It allows for easy automatic installation and updating of packages.

Remote Management Scripts

Checks and tasks that run on the Remote Management platform.

Checks are divided into these categories:

  • Alert - Monitor various bits of a workstation and fail should they detect an issue.
  • Information - Gather information about a workstation and display it. These do not fail during normal operation.

Tools are exactly the same as Checks, but run as Background Tasks instead. This allows them to have a maximum 24 hour runtime, rather than the 150 second maximum for checks.
